Everything You Need

Powerful features packed into a single lightweight script

window

Visual Interface

Displays all available package updates in a native Windows checklist — no command-line knowledge required.

select_all

Batch Updates

Select multiple packages and update them all in a single action. Sit back while Winget-UI handles the rest.

checklist

Easy Selection

"Select All" and "Clear Selection" buttons make managing a long list of updates fast and painless.

security

Auto-Elevation

Automatically requests Administrator privileges via UAC so updates install reliably without extra steps.

upgrade

Winget Version Check

Detects your installed winget version and prompts you to update it before running, keeping the tool chain fresh.

terminal

Run from the Web

No download needed. A single PowerShell one-liner fetches and runs the latest version directly from the internet.

How It Works

Simple, powerful, and built for everyone

1
play_circle

Run the Script

Paste the one-liner into PowerShell or run .\winget-ui.ps1 locally. UAC handles elevation automatically.

2
search

View Updates

Winget-UI scans your system with winget upgrade and presents every available update in a clean GUI window.

3
checklist_rtl

Select Packages

Check the applications you want to update. Use "Select All" to grab everything, or pick and choose.

4
rocket_launch

Update & Done

Click Update Selected. The script runs each update sequentially and reports success or failure in the console.
